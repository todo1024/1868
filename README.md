课程介绍：

后台服务器开发，经典的分层是网络层、服务层以及存储层，作为存储层常见的关系型数据库，MySQL在大厂面试以及开发工作中都是必备技能

本课程从MySQL的环境搭建讲起，覆盖MySQL的SQL和索引优化，索引实现原理，存储引擎，事务处理，事务隔离级别以及底层锁机制，中间件Mycat，主从复制和读写分离应用实践等
